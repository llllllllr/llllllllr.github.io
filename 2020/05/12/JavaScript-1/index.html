对象

数字，字符串，布尔值，null, undefined其它所有值都是对象

  • 对象字面量： 包含在一对花括号里面的零或多个“名/值”

  • 检索：

    • []后缀中括住字符串表达式 stooge["first name"]
    • 字符串表达式是常数且为合法标识符 flight.departure.IATA

    检索不到返回undefined值

    • 赋值语句 stooge['first-name']='Jerome'
    • 原本没有的话会扩充 stooge['middle-name']='Lester;stooge.nickname='Judy' '

  • 引用

    • 对象通过引用来传递，他们永远不会被拷贝

  • 原型（prototype)

    每个对象都连接到一个原型对象，并且它可以从中继承属性。所有通过字面量创建的对象都连接到 Object.prototype 这个JavaScript中标准的对象。

    原型连接在更新时不起作用的。当我们对某个对象做出改变时，不会触及到该对象的原型。

    another_stooge['first-name'] = 'Harry'

    • 委托

    原型连接只有在检索值的时候才会被用到。如果我们尝试去获取对象的某个属性值，且该对象没有这个属性名，那么JavaScript会尝试从原型对象中获取。终点Object.prototype。 如果最后还是找不到，结果为undefined，这个过程称为委托。

  • 反射

  • typeof typeof flight.number

    原型链中的任何属性也会产生一个值

    typeof flgiht.toString //‘function’

  • hasOwnProperty(不检查原型链)

  • 枚举(Enumeration)

    • for in 遍历对象中的属性名–包括函数和原型中的属性，所以有必要过滤掉不想要的值
    • 属性名出现的顺序不确定。最好的方法是完全避免使用for in语句

  • 删除

    • delete用来删除对象的属性。

  • 减少全局变量污染

  • JavaScript可以很随意地定义那些可保存所有应用资源的全局变量。应使用闭包进行信息隐藏，这是一个有效减少全局污染的方法。

函数

  • 函数对象

在JavaScript中函数就是对象。

  • 函数对象连接到Function.prototype（该原型对象本身连接到Object.prototype)

  • 每个函数对象创建时附加的隐藏属性 1-函数上下文 2-实现函数行为的代码

  • 每个函数对象创建时也随带一个prototype属性，它的值是一个拥有constructor属性且值即为该函数的对象。这和隐藏连接到 Function.prototype完全不同

  • 函数是对象，所以可以存放在变量，对象和数组中

  • 函数字面量

包括：

  • 保留字function。
  • 函数名，可以被省略。
  • 圆括号中的一组参数。
  • 花括号中的一组语句。

闭包

函数字面量可以出现在任何允许表达式出现的地方。函数也可以被定义在其它函数中。一个内部函数自然可以访问自己的参数和变量，同时它也能访问它被嵌套在其中的那个函数的参数与变量，通过函数字面量创建的函数对象包含一个连到外部上下文的连接。这被称为闭包。

JavaScript中的四种调用模式：

  • 方法调用模式

  • 函数调用模式

  • 构造器调用模式

  • apply调用模式

    apply方法让我们构建一个参数数组并用其去调用函数。它也允许我们选择this的值。

  • 参数

当函数被调用时，会得到一个arguments数组，使得编写一个无需指定参数个数的函数称为可能。

    throw 语句抛出一个exception对象，该对象包含可识别异常类型的name属性和一个描述性的message属性。

  • 给类型添加方法

    通过给Object.prototype添加方法使得该方法对所有对象可用。

    👇通过给Function.prototype增加一个method方法，我们就不必键入 prototype这个属性名。

    • Number.prototype添加一个integer方法

    • String.prototype添加一个trim方法

    给基本类型添加方法，我们可以大大地提高语言的表现力。因为JavaScript原型继承的动态本质，新的方法立刻被赋予到所有的值（对象实例）上，哪怕值（对象实例）是在方法创建之前就被创建好了。

    • 变量的作用域（全局变量和局部变量）

      链式作用域结构，子对象会一级一级地向上寻找所有父对象的变量。所以父对象的所有变量对子对象都是可见的，反之不成立。

  • 模块

    可以使用函数和闭包来构造模块。模块是一个提供接口却隐藏状态与实现的函数或对象。
