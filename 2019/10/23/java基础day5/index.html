小结

wait();

notify();

notifyAll();

都使用在同步中

  • 因为要对持有监视器（锁）的线程进行操作，所以要使用在同步中，因为只有同步才有锁
  • 等待和唤醒必须是同一个锁。而锁可以是任意对象，所以可以被任意对象调用的方法定义Object类中。
