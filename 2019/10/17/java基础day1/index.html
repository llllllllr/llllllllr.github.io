内部类

⭐内部类的访问规则

  • 内部类可以直接访问外部类的成员，包括私有成员。

  • 内部类之所以可以直接访问外部类成员，因为它持有了内部类的对象 格式 Outer.this

  • 外部类要访问内部类，必须建立内部类对象。

内部类使用场景

  • 当描述事物时，如果事物的内部还有事物，该事物要用内部类来描述，因为内部事物在使用外部事物的内容。
