停止线程

停止线程—run()方法结束

开启多线程运行，运行代码通常都是循环结构。

只要控制住循环，就可以让run方法结束，也就是让线程结束。

  • 首先t1,t2调用run()方法，但是因为flag=true，它们进入冻结状态，这时候再改变flag=false,它们是无法再判断并进行，线程就不会结束。

如果改为用interrupt()方法，能强制把处于冻结状态的线程t1唤醒，继续执行。
