    StringBuffer(线程安全)

    • 是一个字符串容器
    • 长度是可以变化的
    • 可以操作任意数据类型
    • 可以通过toString()方法变回String类型数据
    • 一旦创建了一个StringBuffer对象，增删改查都在同一个对象上运行（面盆理论）。

    dahelloda

    删除👇

    原来字符串：abcdefg
    后来字符串：adefg

    StringBuilder（不保证同步）

    StringBuilder在功能上和StringBuffer一样，但是不是线程安全的

    也就是说，如果有多个线程同时对容器操作，StringBuffer是会判断锁的

    但是StringBuilder不会，所以StringBuilder不具有线程安全性

    但是，如果在单线程程序里，StringBuilder运行效率更高。
